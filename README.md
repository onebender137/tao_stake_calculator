# Syndicate Staking Calculator

A lightweight, interactive Streamlit dashboard designed to calculate compounding ROI and Dollar Cost Averaging (DCA) for crypto assets like TAO, SOL, and HBAR. 

## Features
* **Live Market Data:** Pulls real-time CAD pricing directly via the free CoinGecko API.
* **Compounding Projections:** Accurately models weekly compounding APY alongside a weekly DCA schedule.
* **Data Visualization:** Generates interactive year-over-year portfolio growth charts and tabular breakdowns.
